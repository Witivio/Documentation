# Supported Clients

## Microsoft Teams clients
Witivio solutions are a suite of web applications and a suite of integrated services accessible from Microsoft Teams.
Depending of the solutions, they can be accessed from the Microsoft Teams desktop client, mobile clients and from the Microsoft Teams web client.
[Get clients for Microsoft Teams](https://docs.microsoft.com/en-us/microsoftteams/get-clients)

## Microsoft 365 Copilot and Copilot Chat

Witivio's AI agents — Agent Control Pro, AI Desk Pro, Diagram Maker Pro, GPT Pro, Mail Pro, and Manager Insight Pro — are designed to run inside **Microsoft 365 Copilot** and **Microsoft 365 Copilot Chat**.

::: tip
**Copilot Chat** is automatically included with any Microsoft 365 subscription and is grounded in the web. **Microsoft 365 Copilot** is an added per-user license that extends Copilot Chat with chat grounded in your organization's data through Microsoft Graph. Witivio agents work with both.
[Learn more about the difference](https://learn.microsoft.com/microsoft-365/copilot/which-copilot-for-your-organization).
:::

Users sign in with their Microsoft Entra work or school account and can access Witivio agents from any of the following Microsoft-supported entry points:

- **Microsoft 365 Copilot app** — web, Windows desktop, and mobile (iOS, Android)
- **Microsoft 365 Copilot on the web** — `https://m365copilot.com` (redirects to `https://m365.cloud.microsoft/chat`)
- **Microsoft Teams** — desktop and web client
- **Microsoft Outlook** — desktop and web
- **Microsoft Edge** — Copilot Chat side pane

For the authoritative and continually updated list of Copilot entry points and eligibility, see the Microsoft documentation:
[Manage Microsoft 365 Copilot Chat](https://learn.microsoft.com/copilot/manage) and
[Overview of Microsoft 365 Copilot Chat](https://learn.microsoft.com/copilot/overview).
