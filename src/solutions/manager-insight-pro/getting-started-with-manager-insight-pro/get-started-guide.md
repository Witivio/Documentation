# Manager Insight Pro - Get Started Guide 

Manager Insight Pro is an AI-powered management assistant available directly inside Microsoft Copilot and Microsoft Teams.
It helps managers get visibility on their team, projects, and key metrics through natural, intelligent conversations — without leaving their workspace.
Manager Insight Pro provides read-only insights based on your permissions. It never modifies data.
Manager Insight Pro may require activation or admin consent depending on your organization's setup. Once approved, it becomes available for authorized users.

---

## Installation & Access (via Microsoft Teams or Copilot)

Manager Insight Pro is available inside both **Microsoft Copilot** and **Microsoft Teams**.

How to access Manager Insight Pro
1. Open Microsoft Copilot
2. Go to Copilot Agents
3. Select Manager Insight Pro
4. Start asking your questions
Or via Microsoft Teams:
1. Open Microsoft Teams
2. Go to Apps
3. Search for Manager Insight Pro
4. Open the application with copilot
If Manager Insight Pro does not appear, your administrator may need to enable it in the Teams Admin Center.

## Activation (Admin Consent)

Depending on your organization, Manager Insight Pro may require **one-time activation** or admin consent to access management data.

If activation has not been completed, you may see a message indicating that activation is required.
Admin consent allows Manager Insight Pro to use the necessary Microsoft Graph permissions to provide insights.

Contact your IT administrator or Microsoft 365 administrator to complete activation if needed.
