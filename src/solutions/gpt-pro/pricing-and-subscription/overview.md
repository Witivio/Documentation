# Pricing & Subscription 

 

## Usage Limits & Message Policy 

 

GPT Pro uses a **rolling 3-hour usage model**. 

 

### Message counting 

 

**1 message = 1 user message OR 1 GPT Pro response** 

 

Examples: 

- You send 5 messages + GPT replies 5 times → **10 messages counted** 

- Asking for 3 variations → **3 messages counted** 

 

### Rolling 3-hour window 

- Each plan includes a specific message allowance 

- Usage refreshes continuously over a **3-hour sliding window** 

- When the limit is reached: 

  - GPT Pro pauses temporarily 

  - The user may wait or upgrade the plan 

 

Full details on <a href="https://marketplace.microsoft.com/en-us/product/saas/witivio.gpt-pro?tab=PlansAndPrice" target="_blank" rel="noopener">Microsoft Marketplace</a>

## Pricing 

 

Pricing is available through **in-app purchase** in the Microsoft Teams Store.   

GPT Pro uses **User – Hourly Usage Plans** (Starter, Premium, Platinum). 

 
Full pricing details on <a href="https://www.teams-pro.com/en/gpt-pro#pricing" target="_blank" rel="noopener">TeamsPro website</a>
