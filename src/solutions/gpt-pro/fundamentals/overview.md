# Fundamentals 

 

GPT Pro is built around natural-language interactions.   

The app provides guidance through built-in suggestions and supports freeform prompting. 

 

## Prompt Suggestions (built-in) 

GPT Pro includes a suggestion panel displayed at the bottom of the chat window, offering one-click access to common tasks. 
<img src="/assets/img/teams-pro/gpt-pro/gpt-pro-suggestion-panel.png"/>
 

### What can you do? 

Shows what GPT Pro can help you with. 

 

### Help me write a message 

Get help writing or improving any message. 

 

### Summarize this text 

Summarize any text you provide. 

 

### Explain something simply 

Get a simple, clear explanation for any topic. 

 

### Help me organize my day 

Get help planning your day or week. 

 

These suggestions help new users get immediate value without crafting prompts manually.

## Prompt Ideas (for everyday use) 

 

### Writing & Rewriting 

- Rewrite this professionally: … 

- Make this clearer and more concise: … 

- Draft a Teams message (120–150 characters) announcing the update. 

 

### Translation 

- Translate to English, friendly tone: … 

- Translate this into Spanish, professional tone. 

 

### Summaries 

- Summarize this text in 4 bullet points with actions. 

 

### Explanation 

- Explain this simply: … 

- Give me a beginner-friendly definition of … 

 

### Organization 

- Help me plan my day with these tasks: … 

- Suggest priorities based on this list: … 

 

--- 

 

## Prompting Tips 

 

- Be clear about tone, recipient, and goal   

- Mention preferred length (short, concise, long)   

- Ask for alternatives   

- Iterate: “shorter”, “more formal”, “more friendly”   

- Paste drafts directly for improvement   

 

--- 

 

## Best Practices 

 

- Always review the generated text before sending   

- Provide context when the message is important   

- Use GPT Pro to accelerate repetitive writing tasks   

- Ask for adjustments when needed
