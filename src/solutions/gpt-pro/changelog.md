## What's New in This Update

Welcome to the latest version! We're excited to introduce some fantastic improvements:

### Bug Fixes:
1. **Pagination Display:** Fixed the issue causing pagination to display even when there are fewer elements to display.
2. **History Content:** Resolved the problem of raw and unformatted markdown display in the history content.
3. **Task Module Messages:** The messages sent through a task module are now counted as user messages.
4. **Profile Images:** Fixed the incorrect display of user profile images.
5. **Welcome Card Translation:** Corrected the issue of cloning a YouTube link when translating a welcome card.
6. **Pop-up Display:** Fixed the problem of the pop-up choice still being displayed even after it is closed.
7. **Consent Error:** Resolved the 500 error when clicking on Cancel after logging in from a new tenant without consenting to the app.
8. **Prompt Category Display:** The category for the menu is no longer displayed/selected when creating a prompt and selecting shared.
9. **Special Character Display:** Fixed the issue of special characters appearing in the users' frequent request in the dashboard.
10. **Statistics Error:** Resolved the Npgsql.PostgresException on ComputeAiStatisticsAsync in the statistics section.
11. **SharePoint Error:** Fixed the issue of an expired token throwing an exception while trying to refresh it.

### New Features:
1. **GPT Pro Chatbot:** As a user, you can now communicate with a GPT Pro Chatbot in a chat interface on Microsoft Teams.
2. **Conversation Reset:** As an administrator, you can start a new conversation with GPT Pro so you can reset the context of the conversation.
3. **Usage KPIs:** As an administrator, you can track the usage KPIs on your GPT Pro so you can have a better understanding of your collaborators' usage.
4. **Plugin Management:** As an administrator, you can add and remove plugins for your users so you can add additional features depending on your needs.
5. **User Management:** As an administrator, you can add some users into the app so they can manage plugins and access to GPT Pro.
6. **Automatic Deployment:** As a developer, you would like to be able to deploy updates automatically.
7. **Sign In/Up:** As a new customer, you would like to sign in or sign up.
8. **Admin Site Navigation:** As a customer, you would like to be able to navigate in the admin site.
9. **Usage Data Access:** As a customer, you would like to have access to usage data.
10. **Bot Manifest Download:** As a customer, you would like to set up a bot to download the manifest.
11. **Subscription Details:** As a customer, you would like to see your subscription detail.
12. **Document Management:** As an administrator, you can add and remove documents for your users so you can handle their status and modify the list of available documents.
13. **Prompts and Welcome Cards Management:** As an administrator, you can manage and create prompts and welcome cards so your users can have an easier experience chatting with the bot.
14. **Administrator Management:** As an administrator, you can manage the administrators of your licence so you can add/delete other administrators.
