# Q&A

### Does Diagram Maker Pro require a Microsoft 365 Copilot license?
Yes. Diagram Maker Pro is a Declarative Agent for Microsoft 365 Copilot and requires an active Copilot license to function.

### What types of diagrams can Diagram Maker Pro generate?
Diagram Maker Pro supports flowcharts, mind maps, Gantt charts, journey diagrams, quadrant charts, pie charts, and requirement diagrams — all from plain natural language descriptions.

### Do I need to know Mermaid syntax to use Diagram Maker Pro?
No. Diagram Maker Pro generates Mermaid diagrams automatically from your descriptions. No syntax or diagramming skills are required.

### Where are my diagrams stored?
Diagram Maker Pro does not store diagrams on external servers. All output stays within your Microsoft 365 environment. You control where diagrams are saved.

### Can I edit the generated diagram?
Yes. The generated Mermaid code can be copied and edited in any Mermaid-compatible editor or embedded directly into SharePoint, wikis, or documentation tools.

### Does it read the content of my files or emails?
No. Diagram Maker Pro works exclusively from what you describe in your prompt. It does not access your mailbox, files, calendar, or any other Microsoft 365 data source.

### Can my administrator deploy Diagram Maker Pro for the whole organization?
Yes. Diagram Maker Pro can be deployed organization-wide via the Microsoft 365 Admin Center. Admin consent may be required depending on your tenant configuration.

### What happens if the generated diagram is not accurate?
You can follow up with a refinement prompt — for example: "simplify this", "add a step between X and Y", or "switch to a Gantt chart". Diagram Maker Pro iterates based on your feedback.
