## Recurrent events
Schedule events that repeat regularly with ease. Set the recurrence of the event (e.g., daily, weekly, monthly) using the "Recurrence" dropdown menu.

<ol>
    <li>Open the event form and select <strong>Recurrence</strong>.</li>
    <li>Choose the pattern (e.g., <strong>Daily</strong>, <strong>Weekly</strong>, <strong>Monthly</strong>).</li>
    <li>Confirm and <strong>Save</strong>.</li>
  </ol>

## Files
Attach relevant files to your event. Click on the "Add" button to upload files from your device.

 <ol>
    <li>Click <strong>Add</strong> in the event form to attach files.</li>
    <li>Select the file from your device.</li>
    <li><strong>Save</strong> the event to confirm the attachment.</li>
  </ol>
  <p>Files are stored in your organizationâ€™s <strong>Microsoft 365</strong> (Teams <strong>SharePoint/OneDrive</strong>).</p>
