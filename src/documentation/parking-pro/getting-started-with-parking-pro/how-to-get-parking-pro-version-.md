<p class="no-margin">Click on the <b>Help button ❔</b> next to the Settings ⚙️ button &gt; <b>select ' ℹ️ About' (1)</b>.</p>
<p class="no-margin"></p>
<p class="no-margin">A new dialog will pop up with the <b>Parking Pro 'Version' (2)</b></p>
